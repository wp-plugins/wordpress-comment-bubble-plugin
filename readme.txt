== Description ==

This is easy plugin to add comment bubble count to each post title. Add this feature to your blog post titles by displaying the total count of the comments for that post. Not every themes has this feature included by default, here is the fastest way to do it! It is use CSS button without any image.

The comment bubble shown only on front page post and default status is on when plugin activated.

[Live Demo](http://patrick.bloggles.info/)

== Installation ==

= The Plugin can be installed directly from the WordPress Plugin page =

1. Go to the Plugins => Add New.
2. Enter `Wordpress Comment Bubble` (without quotes) in the textbox and click the `Search Plugins`.
3. In the list of relevant Plugins click the `Install` link on the right hand side of the page.
4. Click the `Install Now` button on the popup window.
5. Click `Activate Plugin` to finish installation.
6. That's it!

= Download and upload =

1. Upload `comment-bubble.php` to your `/wp-content/plugins/ directory`.
2. Activate the plugin through the `Plugins` menu in WordPress
3. Setting on Appearance Menu.

Support WordPress and WordPress Multisite
